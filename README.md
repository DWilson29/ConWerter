# ConWerter - A tool for learning and converting CW (Morse Code)
## Introduction
ConWerter is a simple GUI tool designed to convert text into morse code, or vice versa, by either entering the dits and dahs manually or (eventually) with ML identification of a provided sound file. 

## Installation
No binaries are currently provided. No installer is currently provided. Currently, it only supports Windows, but (eventually) will support MacOS/Linux.

### Compilation
To compile the project, you simply need
1. .NET 10 SDK from https://dotnet.microsoft.com/en-us/download/dotnet/10.0
2. Install Avalonia UI from https://avaloniaui.net

Then you can simply compile it using `dotnet publish` to generate the executable.
